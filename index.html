  • An efficient approximate version of KDE.
  • Non-memory based.

    • Random Fourier features to approximate the Gaussian kernel.
    • Density Matrices to represent the PDF.

Second Phase: Density Matrix Initialization

  • \[\begin{equation} \rho_t = \frac{1}{n}\sum_{i=1}^N q_i \cdot \phi_\text{aff}(\bf{x_i})\phi^t_\text{aff}(\bf{x_i}) \end{equation} \]
    where $\sum_{i=1}^{t} q_i = 1$
  • An initial portion of the stream $\{\bf{x}_1, \cdots, \bf{x}_n\}$ is used to update the density matrix.
  • Only normal data points are used in this phase.

Third Phase: Quantum Measurement

  • \[\begin{equation} \hat{f}(\bf{x}_{t+1}) = \frac{1}{M_\sigma} {\phi}_\text{aff}(\bf{x}_{t+1})^T \rho_t \: {\phi}_\text{aff}(\bf{x}) \end{equation} \]
    where $\rho$ is the density matrix defined in the last phase and $M_\sigma$ is a normalization constant.

Fourth Phase: Anomaly Detection Classification

  • \[\begin{equation} \hat{y}(\bf{x}_{i}) = \left\{\begin{matrix} \text{ `normal'} & \textit{if } \hat{f}(\bf{x}_{i}) \geq \tau\\ \text{ `anomaly'} & \textit{otherwise} \end{matrix}\right. \end{equation} \]

Fourth Phase: Anomaly Detection Classification

  • \[\begin{equation} \hat{y}(\bf{x}_{i}) = \left\{\begin{matrix} \text{ `normal'} & \textit{if } \hat{f}(\bf{x}_{i}) \geq \tau\\ \text{ `anomaly'} & \textit{otherwise} \end{matrix}\right. \end{equation} \] The threshold $\tau$ is found using two different approaches.
  • Prior knowledge of the proportion of anomalies ($\beta$).
  • Use an optimization metric computed over $\{\hat{f}(\bf{x_1}), \cdots, \hat{f}(\bf{x_n})\}$, for instance using the best threshold with respect to AUC-ROC in initial memory..

Fifth Phase: Density Matrix Update

  • \[\begin{equation} \rho_{t+1} = (1-\alpha) \cdot \rho_{t} + \alpha \cdot \phi_\text{aff}({\bf{x}_{t+1}})\phi^T_\text{aff}({\bf{x}_{t+1}}) \end{equation} \]

Conclusions

  • InQMAD show better performance than state-of-the-art algorithms over twelve anomaly detection data sets.
  • It computes density estimation given a normality measure for each point.
  • As future work, we will continue to further develop the main concepts behind InQMAD, building algorithms based on the combination of Fourier features and density matrices with deeper neural networks, such as autoencoders.
